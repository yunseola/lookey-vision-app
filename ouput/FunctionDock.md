### 상세 기능

#### 📍 편의점 구조 안내
- **프로세스**
  1. 메인 화면에서 “편의점” 버튼 클릭  
  2. 카메라 화면 실행  
  3. 사용자가 휴대폰을 비춘 방향으로 탐색  
  4. 이동 가능 여부를 음성 안내 (예: “정면으로 이동 가능합니다.”)  
  5. 매대가 인식되면 “과자 매대가 있습니다.” / 인식 불가 시 “전방에 매대가 있습니다.”  
  6. 계산대가 있으면 “정면에 계산대가 있습니다.”  

---

#### 📍 계산대 위치 안내
- **프로세스**
  1. 메인 화면 → “편의점” 버튼 → “길 안내” 버튼  
  2. 카메라 실행 후 계산대 탐색  
  3. 화면에 잡히면 “정면에 계산대가 있습니다.” 음성 출력  
  4. 계속 잡히면 반복 안내  

---

#### 📍 장애물 알림

- **프로세스**
  1. 메인 화면 → “편의점” 버튼 → “길 안내” 버튼  
  2. 카메라 실행  
  3. 장애물 인식 시 음성 안내  

---

#### 📍 근처 편의점 안내
- **프로세스**
  1. 홈 화면 → “편의점 안내” 버튼  
  2. 가까운 편의점 3곳을 거리순으로 제공 (매장명, 거리 정보 포함)  
  3. 사용자 선택 (터치 + TalkBack 지원)  
  4. 해당 매장까지 길 안내 시작  

---

#### 📍 길 안내
- **프로세스**
  - 지도 API 연동하여 길찾기 제공

## 상품

### 📍 상품 인식
- **프로세스**
  1. 메인 화면 → “편의점” 버튼 → “상품 인식” 버튼
  2. 카메라 실행 (광각 카메라 우선, 안 되면 버튼으로 전환)
  3. 위 → 아래 스캔  
     - 버튼 누르고 3초간 촬영  
     - 음성 안내: “위에서 아래로 천천히 휴대폰을 움직여주세요”
  4. 촬영 이미지 분석 (AI + OCR + 이미지 분류)
  5. 결과 안내  
     - 매칭 성공: 상품명/가격/행사 여부 + 알레르기 정보 음성 안내  
     - 매칭 실패: “해당 상품이 이 매대에 없습니다. 옆으로 이동해 스캔해주세요.”
  6. 스캔 종료 시 자동으로 일반 카메라 모드 전환 (수동 버튼도 가능)
  7. 화면에 잡힌 단일 상품은 실시간 음성 안내  
     - “코카콜라 제로 500ml, 가격 2200원, 행사 1+1입니다.”  
     - 장바구니에 있던 상품이면: “장바구니에서 제거할까요?” (모달 표시)

---

### 📍 상품 성분 분석
- **프로세스**
  1. 상품 인식 후 성분 OCR 수행
  2. 사용자 알레르기 정보와 매칭
  3. 주의 성분 포함 시 음성 안내  
     - “이 상품에는 유당 성분이 포함되어 있습니다. 주의하세요.”

---

### 📍 상품 행사 여부
- **프로세스**
  1. 상품 인식 시 행사 여부 확인
  2. 음성으로 안내  
     - “현재 2+1 행사 상품입니다.”
  3. 장바구니 담기 시 행사 조건 자동 반영

---

### 📍 상품 위치 / 존재 탐색
- **프로세스**
  1. 상품 검색(음성/텍스트 입력)
  2. 매대 스캔 후 인식 여부 확인
     - 있으면 1초 단위로 위치 안내 (정중앙 기준)
     - 없으면 “해당 상품이 현재 이 매대에 없습니다” 안내
  3. 필요 시 옆 매대로 이동 후 재스캔

---

### 📍 장바구니 담기 / 삭제
- **프로세스**
  1. 메인 화면 → “장바구니” 버튼
  2. 음성 검색 또는 상품 인식 모드에서 직접 담기
  3. 담기/삭제 시 음성 안내  
     - “코카콜라 제로를 장바구니에 담았습니다.”  
     - “장바구니에서 제거할까요?” (모달 제공)
  4. 모든 상품 선택 후 확인 → 총액 및 행사 적용 가격 안내
  5. 장바구니 상품 터치 시 해당 행사 정보 음성 안내

| 요구사항명       | 기능명                 | 기능 상세 | 우선순위 |

| 알레르기 정보 입력 | 알레르기 정보 입력       | 유저가 가진 알레르기 정보를 입력한다. | 1순위 |

## 회원

### 📍 회원가입 및 로그인
- **프로세스**
  1. 메인 화면 → “구글 회원가입” 버튼 클릭  
  2. 기기 내 구글 계정 선택  
  3. 선택된 계정으로 회원가입 및 로그인 완료  

---

### 📍 알레르기 정보 입력
- **프로세스**
  1. 메인 화면 → “알레르기” 버튼 클릭  
  3. “+” 버튼을 눌러 알레르기 항목 입력  
  4. 입력 완료 후 “확인” 버튼 클릭  
  5. 등록된 알레르기 정보가 저장되어 상품 분석 시 매칭됨
